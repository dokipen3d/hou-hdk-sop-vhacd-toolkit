= Engine (v-hacd) =

""" Generates convex hulls by using V-HACD decomposition """

@inputs

Geometry:
	All incoming geometry. By default node will error out if it detects geometry other than polygons.

@parameters

== Filter ==

Allow Parameters Override:
    #channels: /allowparametersoverride
    Turn ON/OFF possibility to override parameter values with attributes found on incoming geometry

Convert To Polygons:
    #channels: /converttopolygons
    Convert all incoming geometry to polygons
	
== Main ==

Mode:
    #channels: /decompositionmode
    0: Voxel based decomposition 1: Tetrahedron based decomposition
