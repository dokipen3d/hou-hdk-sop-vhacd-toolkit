= Scout Sr. (v-hacd) =

""" Generate some additional data per convex hull/bundle. """

Use this node to add/create additional data that other nodes in V-HACD toolkit may require.
To use all available parameters, this node requires ((bundle_id)) primitive attribute.
By default it will only set node warning when ((bundle_id)) primitive attribute is not found on one of the inputs.

@parameters

== Filter ==

Missing Bundle ID:
	#id: missingbundleiderrorlevel
	Specify error level when ((bundle_id)) attribute is not found on one of the inputs.
	:note:
	Node will override this behaviour and set error state when user will use any parameter that requires ((bundle_id)) attribute.
	
	None:
		#id: 0
		Ignore when ((bundle_id)) is missing.

	Warning:
		#id: 1
		Raise warning when ((bundle_id)) is missing.

	Error:
		#id: 2
		Raise error when ((bundle_id)) is missing.
		
== Main ==
			
Add Hull Count ATT:
	#id: addhullcountattribute
	Add ((hull_count)) detail attribute that stores information about how many convex hulls were detected.

Add Hull ID ATT:
	#id: addhullidattribute
	Add ((hull_id)) primitive attribute that helps to identify to which convex hull polygons belongs.

GRP Per Hull:
	#id: groupperhull
	Create primitive group per each detected convex hull.

	Name:
		#id: specifyhullgroupname
		Specify convex hulls group partial name.
		
Add Bundle Count ATT:
	#id: addbundlecountattribute
	Add ((bundle_count)) primitive attribute that stores information about how many bundles were detected.
	:note:
	Requires ((bundle_id)) attribute.
	
GRP Per Bundle:
	#id: groupperbundle
	Create primitive group per each detected convex bundle.	
	:note:
	Requires ((bundle_id)) attribute.
	
	Name:
		#id: specifybundlegroupname
		Specify bundle group partial name.
